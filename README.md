## How GitHub Sync works

1. **Connect your GitHub account** on your profile page.
2. **Auto-commit** — every time you submit a solution, it is pushed here automatically.
3. **Bulk Sync** — push all your past solutions at once from the GitHub settings page.

## Repository structure

Solutions are organized by difficulty, then problem title. Each submission is stored as a separate file:

```

```

**Example:**
```
Easy/Two-Sum.md
Medium/Add-Two-Numbers.md
```
